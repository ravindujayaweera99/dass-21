# DASS-21 Web App (Vanilla JS)

This folder contains a static web version of your Shiny (R) DASS-21 app, preserving:
- **Wordings** (Sinhala & English)
- **Colors / styling**
- **Question flow (pages 0 → 5)**

## How to run (no server required)
Open `index.html` in a browser.

> If your browser blocks local iframes or fonts when opening from file://,
> run a tiny local server:
> - Python: `python3 -m http.server 8000`
> - Then open: `http://localhost:8000`

## Assets you must place
Copy these files into `assets/` (same names):
- `logo.jpg`
- `download2.jpg`

## “Download PDF” button
In this static version, the **Download PDF / PDF ලෙස බාගන්න** button triggers the browser print dialog (you can “Save as PDF”).
